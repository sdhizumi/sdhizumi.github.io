Tiny Bleeps

2023/04/30 - M3-2023 Spring

Experience minimal bleep loop music made with source files limited to 1266 bytes!

This album was made with FAMIC on USB.

The source files (MML) are included as a bonus for people who purchase the album. (They are not included for single track purchases.)

What is FAMIC on USB?

FAMIC on USB is a melody IC that can be written in MML. It was created by Prioris LLC. in Japan. It has sine, pulse, triangle, saw, 1-bit noise, and wave ram sound generators. Like with the FC/NES and GB, you can write any MML sequences up to the 1266-byte limit.

The editor MML Playground linked below works on modern browsers like Firefox, Chrome, etc., so you can try MML writing for free. Data made with this editor can be written to and played on real hardware. famic.prioris.jp/mml/

More information is available on the official site (Japanese only). famic.prioris.jp
